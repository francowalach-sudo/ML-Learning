## Python coding
### Basics
**How to define variables:**

*Int and float:*

*To define an integer, use the following syntax

<Variable_name> = any Natural number

*To define a float (number with decimals), you can use any of the below syntax:

<Variable_name> = any number with decimals

<Variable_name> = float(Natural Number)

*String*

*To define a string you can do it either with a single quote or a double quotes

<Variable_name> = "Any text you want"

<Variable_name> = 'Any text you want'
